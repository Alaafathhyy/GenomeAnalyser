# BioServer
 A website for applying a biological functions on DNA sequence

## Functions include:-
* Translate 
* Transcribe
* GC
* Melting Temprature 
* Reverse
* Reverse Complement
* Complement 
* Get Data Base Statistical 
* Show the Data Base
* Update personal info
* Upload Fasta file

## Tools and languages that used :
* Apache Server which support 
   * MySql
   * Php Server
* Html
* java Script
* CSS
